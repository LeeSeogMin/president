# 부록 A. 분석 방법론 상세

## A.1 Adaptive Capacity Wheel (ACW) 프레임워크

Gupta et al.(2010)이 기후변화 적응 역량 평가를 위해 개발한 ACW는 6개 차원, 22개 하위 기준으로 구성된다. 본 분석에서는 이를 정부의 일반적 적응 역량 평가로 확장 적용하였으며, 이 확장은 Grothmann et al.(2013)과 Pahl-Wostl(2009)이 적응 역량 프레임워크의 다영역 적용 가능성을 논의한 바에 근거한다.

### A.1.1 6개 차원 및 22개 기준

<부표 A-1> ACW 22개 기준 목록

| 차원 | 코드 | 기준명 | 평가 내용 |
|------|------|--------|----------|
| 다양성(Variety) | V1 | 문제 프레이밍 다양성 | 정책 문제를 다양한 관점에서 정의하는 능력 |
| | V2 | 다수준 거버넌스 | 중앙-지방-국제 수준 간 정책 조율 |
| | V3 | 다부문 접근 | 부처·분야 간 통합적 접근 |
| | V4 | 솔루션 다양성 | 정책 대안의 다양성과 유연성 |
| 학습역량(Learning) | Le1 | 비전 제시 | 적응적 비전의 명확성과 전달력 |
| | Le2 | 기업가적 리더십 | 혁신적 정책 추진 의지와 실행력 |
| | Le3 | 협력적 리더십 | 이해관계자 간 협력 촉진 능력 |
| | Le4 | 반응성 | 환경 변화에 대한 정책 대응 속도 |
| 여유자원(Latent) | L1 | 과거 경험 활용 | 선행 정책 경험의 학습·활용 |
| | L2 | 제도적 기억 | 정부 교체에도 지속되는 조직 학습 |
| | L3 | 이중순환학습 | 목표·전략 자체의 재검토 능력 |
| | L4 | 모니터링 | 정책 효과의 체계적 관찰·측정 |
| 리더십(Leadership) | R1 | 연속성 | 정책의 시간적 연속성 확보 |
| | R2 | 자원 가용성 | 필요 자원 동원·배분 능력 |
| | R3 | 혁신 여지 | 새로운 접근을 시도할 수 있는 조직 문화 |
| | R4 | 자율적 변경 권한 | 정책 수정·변경의 자율성 |
| 자원(Resources) | Re1 | 재정 자원 | 정책 집행을 위한 재정 확보 |
| | Re2 | 인적 자원 | 전문인력 확보·역량 |
| | Re3 | 제도적 권한 | 법적·제도적 근거와 권한 |
| 공정거버넌스(Fair Gov.) | F1 | 합법성 | 법적 근거에 기반한 정책 추진 |
| | F2 | 형평성 | 정책 수혜의 공정한 배분 |
| | F3 | 반응성 | 시민 요구에 대한 정책적 반응 |

### A.1.2 점수 척도

| 점수 | 의미 | 판정 기준 |
|:---:|------|----------|
| +2 | 매우 긍정적 | 해당 기준이 체계적으로 구현되어 실질적 성과를 산출 |
| +1 | 긍정적 | 해당 기준이 부분적으로 구현되어 있음 |
| 0 | 중립 | 해당 기준과 관련된 활동이 미미하거나 판단 불가 |
| −1 | 부정적 | 해당 기준에 역행하는 정책·행태가 관찰됨 |
| −2 | 매우 부정적 | 해당 기준에 체계적으로 역행하며 심각한 문제 발생 |

### A.1.3 Gupta et al.(2010) 원 프레임워크와의 차이

| 항목 | 원 프레임워크 | 본 분석의 적용 |
|------|------------|--------------|
| 적용 대상 | 기후변화 적응 제도 | 정부 일반의 적응 역량 |
| 평가 단위 | 단일 제도·법률 | 정부 전체(임기 단위) |
| 평가 방식 | 전문가 패널 질적 평가 | (1) 단일 코더 점수화, (2) 다중 AI 코더 병행 |
| 척도 | 동일(−2~+2) | 동일(Panel 1), 0~+2(Panel 2 sub-wheel) |
| 비교 대상 | 국가 간 비교 | 동일 국가 정부 간 시계열 비교 |

### A.2.3 5라운드 반복 측정

각 모델이 동일 프롬프트에 대해 5회 독립 측정을 수행하였다. 반복 측정의 목적은 (1) 단일 측정의 무작위 편향 제거, (2) 측정 안정성(표준편차) 확인이다.

**프롬프트 설계 원칙**:
- 판단유보(0점) 금지 지침 적용: "정보가 부족하더라도 가용한 근거에 기초하여 방향성 있는 판단을 제시할 것"
- HyperCLOVA HCX-007의 thinking 기능 비활성화(`thinking: {"type": "disabled"}`)로 응답 일관성 향상
- 각 라운드는 독립적 세션에서 실행(이전 라운드 결과 미제공)

### A.2.4 합의 점수 산출

```
합의 점수 = (1/5) Σ_{m=1}^{5} [(1/5) Σ_{r=1}^{5} Score_{m,r}]
```

즉, 모델별 5라운드 평균을 먼저 산출한 후, 5개 모델의 평균을 최종 합의 점수로 사용하였다.

## A.3 추가 분석 방법

### A.3.1 Policy Lag 분석

OECD Agility Framework(2021)에 근거하여 위기 대응 시간을 3단계로 분해 측정하였다.

| 단계 | 정의 | 측정 기준 |
|------|------|----------|
| 인지 지연(Recognition Lag) | 위기 발생~정부의 공식 인지 | 최초 보고·경보 발령일 |
| 결정 지연(Decision Lag) | 공식 인지~대응 정책 결정 | 대책 발표·법령 공포일 |
| 실행 지연(Implementation Lag) | 정책 결정~현장 집행 | 예산 집행·서비스 개시일 |

비교 대상 위기 사례는 (1) 직접 비교 가능성, (2) 위기 유형의 다양성(감염병, 경제, 안전, 기술)을 기준으로 선정하였다.

### A.3.2 제도적 적응 메커니즘 (Argyris & Schon, 1978)

Argyris & Schon(1978)의 조직학습 이론에 기반하여 4개 차원을 평가하였다.

| 차원 | 코드 | 평가 내용 | 척도 |
|------|------|----------|:---:|
| 정책 피드백 제도 | D1 | 정책 평가·환류 체계의 공식성과 실효성 | 0-2 |
| 정책 실험 제도 | D2 | 파일럿·샌드박스·특구 등 실험적 정책 도구 | 0-2 |
| 시민참여 메커니즘 | D3 | 정책 과정에 대한 시민 참여 채널과 실질성 | 0-2 |
| AI/디지털 거버넌스 | D4 | AI·데이터 기반 의사결정 체계의 제도화 | 0-2 |

점수 기준: 0 = 부재, 1 = 부분적/비공식, 2 = 체계적/공식화.

### A.3.3 국정과제 유형 분류 (Lowi, 1972)

Lowi(1972)의 4유형 분류 체계에 Schneider & Ingram(1990)의 정책 도구 관점을 결합하여 국정과제를 분류하였다.

| 유형 | 정의 | 예시 |
|------|------|------|
| 배분정책 | 특정 집단에 자원 배분 | 인프라 투자, 산업 지원 |
| 규제정책 | 행위 제한·통제 | 환경 규제, 안전 기준 |
| 재분배정책 | 자원의 재배분 | 복지 확대, 조세 개혁 |
| 구성정책 | 정부 구조·제도 변경 | 정부조직 개편, 선거 제도 |

추가로 "적응적 정책 도구 비율"을 산출하였다. 적응적 도구는 파일럿·시범사업, 샌드박스, 데이터 기반 정책 설계, 반복적 수정 메커니즘(주기적 재검토 등)을 포함한다.

### A.3.4 삼각검증 설계 (Denzin, 1978)

Denzin(1978)의 삼각검증(triangulation) 원칙에 따라, 5개 분석의 결과 간 수렴·불일치를 체계적으로 검토하였다. 3개 이상의 독립적 분석에서 동일한 방향의 결론이 도출되면 "강건(robust)"으로, 2개 분석에서 일치하면 "부분적 지지"로, 분석 간 상충이 확인되면 원인 분석과 함께 "불일치"로 판정하였다.

## B.3 문재인 정부 (2017-2022) — 3위, ACW +0.54

## C.2 Fleiss' κ 계수 및 해석

다중 코더 간 일치도를 Fleiss' κ로 측정하였다. 5개 모델이 6개 정부에 부여한 ACW 종합 순위의 일치도를 산출한 결과:

**Fleiss' κ = 0.333 (보통 일치, fair agreement)**

| κ 범위 | 해석 | 본 연구 해당 |
|:---:|------|:---:|
| < 0.00 | 일치 없음 | |
| 0.00-0.20 | 약한 일치 | |
| 0.21-0.40 | **보통 일치** | **해당 (κ=0.333)** |
| 0.41-0.60 | 중간 일치 | |
| 0.61-0.80 | 상당 일치 | |
| 0.81-1.00 | 거의 완전 일치 | |

κ=0.333의 해석: 5개 AI 모델은 세부 점수에서 상당한 차이를 보이나, 순위(rank)에서는 구조적으로 일치한다. 상위권(이재명·노무현), 중위권(문재인), 하위권(이명박·윤석열·박근혜)의 3단계 구분에 대해서는 5모델 전원이 일치한다. κ가 "보통 일치"에 그치는 것은 점수 수준(level)의 모델 간 차이(관대한 모델 vs 엄격한 모델)에 기인하며, 방향성(direction)과 순위(rank)의 일치도는 κ가 반영하는 것보다 높다.
