## 2. 핵심 의사결정: 논문 분리 (2026-03-02)

### 결정 사항

**paper_final_v6.md(통합 논문)를 두 편의 독립 논문으로 분리한다.**

### 분리 근거 (4개 관점 병렬 조사 결과)

#### (1) 학술적 선례 — 분리가 확립된 전통
- Ostrom(IAD), Sabatier(ACF), Kingdon(MSF) 등 행정학 핵심 프레임워크 모두 "이론 제안 → 별도 실증 적용" 경로
- Salami slicing에 **해당하지 않음**: 연구 질문, 방법론, 학술 기여가 각각 상이
- 한국행정학보 분량 제한(27,000자 원칙, 35,000자 상한) 내에서 이론+실증 모두 충실히 전개 불가

#### (2) 현재 논문의 구조적 긴장
- §3의 정량 분석(ACW, Policy Lag, INST)이 SAPD 아키텍처 설계에 실질적으로 반영되는 지점이 거의 없음
- §7.2 역매핑은 정성적 사례 기술만으로도 성립 — §3의 6개 정부 비교에서 단 1개 데이터 포인트만 활용
- 5개 방법론×6개 정부×5개 AI 모델의 분석이 59행으로 압축 → 핵심 내용이 부록으로 밀림
- SAPD 이론(§2,4,5,6)=35%, 실증 분석(§3)=15% — 실증이 방법론적 야심 대비 과소 전개

#### (3) 독자층 분리
- SAPD 이론: 정책설계론, 알고리즘 거버넌스, AI-정책 통합 연구자
- 정부 비교: 한국 행정학, 비교 정부론, 정부 혁신 연구자
- 교집합이 매우 좁음

#### (4) 다중 AI 스코어링의 독립적 학술 가치
- LLM-as-coder가 Political Analysis, AJPS 등에 게재 (2024-2025)
- 5개 LLM으로 정부 제도적 역량을 질적 점수화한 사례는 선행연구 미존재 → 방법론적 기여 확인
- 한국 행정학계에서 LLM-as-coder 최초 시도

## 3. 두 논문 구성안

### 논문 1: SAPD 이론 논문

| 항목 | 내용 |
|------|------|
| **제목(안)** | 상태 기반 적응형 정책 설계(SAPD): 증거 기반 정책의 구조적 한계를 넘어서는 AI 시대 정책 아키텍처 |
| **연구 질문** | EBP의 한계를 극복하는 정책 설계 아키텍처는 어떻게 구성되는가? |
| **v6 원천** | §1, 2, 4, 5, 6, 7, 8, 9 |
| **방법론** | 개념적/이론적 분석 (Jaakkola 2020) |
| **이론 틀** | EBP, 시스템 역학, 사이버네틱스, 알고리즘 거버넌스 |
| **실증** | 정성적 사례만 (v5 수준) — 코로나19, 재난기본소득, AI법 간략 서술 |
| **핵심 기여** | SAP 패러다임 + SAPD 5단계 아키텍처 + 공공가치 내재화 + "리더십→아키텍처" 전환 명제 |
| **예상 분량** | ~26,500자 |
| **투고 대상** | 한국행정학보 |
| **투고 순서** | 2순위 (논문 2 투고 후) |

**§3 제거 후 보완 방식**:
- 서론에서 이재명 정부를 1-2단락 수준 정성 서술 (동기 부여)
- 7.2절 역매핑: 정성적 관찰 기반 + "별도 실증 연구(이석민, 2026b)에서 확인" 교차 참조
- 8.7절의 "다중 AI 채점 신뢰도" 반론 삭제 (논문 2로 이관)

### 논문 2: 이재명 정부 적응역량 실증 논문

| 항목 | 내용 |
|------|------|
| **제목(안)** | 이재명 정부의 적응적 거버넌스 역량 평가: 다중 AI 스코어링에 의한 역대 정부 비교 분석 |
| **연구 질문** | 이재명 정부의 적응역량은 어떤 수준이며, 어떤 구조적 특성을 갖는가? |
| **분석 초점** | 이재명 정부 역량 평가가 주(主), 역대 정부 비교는 맥락/벤치마크 |
| **v6 원천** | §3 + 부록 A-D |
| **방법론** | 다중 방법론 실증 + Multi-LLM 코딩 (5모델×5라운드) |
| **이론 틀** | ACW(Gupta 2010), 적응적 거버넌스(Folke 2005), 조직학습(Argyris & Schon 1978) — SAPD 불필요 |
| **핵심 기여** | (1) 다중 AI 스코어링 방법론, (2) 이재명 정부 적응역량의 체계적 평가, (3) 리더십-제도 괴리 발견, (4) 성남→경기→대통령 스케일업 궤적 분석 |
| **예상 분량** | ~25,000자 |
| **투고 대상** | 한국정책학회보 / 정보화정책 |
| **투고 순서** | 1순위 (먼저 투고) |
