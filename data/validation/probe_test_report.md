- **effective** (식별률 ≤30%): 블라인딩이 효과적. 논문에서 bias-mitigation 효과 주장 가능
- **partial** (30% < 식별률 ≤ 60%): 부분적 블라인딩. 편향 완화 시도로 기록, 효과 주장 약화

## 블라인딩 실패의 원인 분석

블라인딩이 100% 실패한 원인은 **사건의 구조적 고유성**에 있다. 대통령명·정당명·연도를 치환하더라도, 위기 사례의 수치적 디테일(사망자 수, 탑승자 수, 대응 시간)이 사실상 fingerprint로 작동한다:

- "여객선 침몰 + 476명 탑승 + 304명 사망" → 세월호는 전 세계에서 유일한 사례
- "리먼브라더스 파산" → 2008년 글로벌 금융위기로 즉시 특정
- "핼러윈 행사 압사 + 159명 사망" → 이태원 참사로 즉시 특정

LLM의 사전학습 데이터에 이러한 사건이 고유하게 기록되어 있으므로, 이름과 날짜를 치환하는 Level 2 블라인딩만으로는 정부 식별을 억제할 수 없다.

## LLM 정치적 편향이 결과에 미치는 영향: 객관적 평가

### 편향 가능성이 존재한다는 근거

1. **학술 문헌이 확인한 LLM 좌파 성향**: 다수 연구가 ChatGPT의 left-libertarian 성향을 보고함 (Hartmann et al., 2023; Motoki et al., 2024). Nature(2025)는 "LLM은 창조자의 이념을 반영한다"고 보고.

2. **블라인딩 완전 실패**: 5개 모델 모두 정부를 100% 정확 식별 → 사전학습 데이터의 사후적 평가("세월호=실패", "K-방역=성공")가 채점에 반영될 경로가 열려 있음.

3. **5개 모델 전원 동일 방향**: 진보 정부(노무현·문재인)에 전원 양(+), 보수 정부(박근혜·윤석열)에 전원 음(-). 공유 학습 데이터에서 비롯된 집단적 편향 가능성.

4. **진보-보수 격차가 모델에 무관하게 일정**: GPT(+0.84) ~ Grok(+1.26)로 격차 크기는 다르나 방향은 5개 모델 전부 동일.

### 편향이 결과를 전복시키지 않는다는 근거

1. **LLM이 채점하지 않은 지표도 동일 패턴**:
   - T/I/D/E 가중합: 연구자 1인 수동 코딩 → 문재인 +18, 윤석열 -7 (LLM 무관)
   - WGI 잔차: World Bank 공개 데이터 → 노무현 +0.308, 박근혜 -0.086 (LLM 무관)
   - LLM이 관여하지 않은 2개 지표에서도 동일 패턴이면, LLM 편향만으로 결과를 설명할 수 없음

2. **하위 정부의 저평가가 사실에 기반**:
   - 박근혜: 세월호 7시간 의사결정 공백, 국정농단 탄핵 → 헌법재판소 확정 사실
   - 윤석열: 비상계엄 선포(2024.12.03), 이태원 특별법 거부권 570일 → 헌법재판소 확정 사실
   - 이는 "편향"이 아니라 사실의 반영

3. **보수 정부도 높은 점수를 받는 항목 존재**:
   - 이명박: CRS 3.40 (금융위기 1일 만에 대응회의) — 5개 정부 중 상위권
   - 이명박: ACW에서 5개 모델 전원 양(+) — 진보 정부와 동일 방향
   - 윤석열: WGI 잔차 양(+0.030), Lowi STRICT 0.8% → 일부 지표에서 진보보다 높음

4. **편향의 "방향"과 "크기" 구분 필요**:
   - 방향(진보 > 보수)은 편향 가능성이 있으나, 크기의 상당 부분은 사실 기반
   - 문재인이 윤석열보다 높은 것은 편향 없이도 설명 가능 (코로나 당일 대응 vs 비상계엄)

### 지표별 LLM 편향 노출 정도

| 지표 | 채점 주체 | LLM 편향 영향 |
|------|----------|:------------:|
| T/I/D/E 가중합 | 연구자 수동 코딩 | **없음** |
| AGLI | AI 직접 채점 | **가능** |
| Lowi STRICT | 연구자 코딩 + AI 보조 | 제한적 |
| CRS | AI 직접 채점 + 블라인딩 무효 | **가능** |
| WGI 잔차 | World Bank 공개 데이터 | **없음** |

→ LLM 편향이 영향을 줄 수 있는 지표: 2/5 (AGLI, CRS)
→ LLM 편향과 무관한 지표: 2/5 (T/I/D/E, WGI)

### 논문에서의 처리 방향

| 구분 | 내용 |
|------|------|
| **인정해야 할 것** | LLM이 정부를 식별할 수 있으므로, 사전학습 데이터의 사후적 합의가 AGLI·CRS 채점에 반영되었을 가능성을 배제할 수 없다 |
| **주장할 수 있는 것** | LLM이 관여하지 않은 지표(T/I/D/E, WGI)에서도 동일 패턴이 나타나므로, 결과의 방향성이 LLM 편향만으로 설명되지 않는다 |
| **주장할 수 없는 것** | 블라인딩이 편향을 통제했다 |

### 논문 방어 논리 (권고)

> "5개 지표 중 LLM이 직접 채점한 것은 2개(AGLI, CRS)이며, 연구자 코딩(T/I/D/E)과 외부 데이터(WGI)에서도 동일 패턴이 관찰된다. 따라서 LLM 편향이 결과의 방향성을 생성한 것이 아니라, 기존 패턴을 **증폭**했을 가능성은 있으나 **전복**시키지는 않았다고 판단한다."
