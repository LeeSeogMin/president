> 프로젝트: president/

## 1. 프로젝트 개요

6개 한국 대통령 정부(노무현~이재명)의 적응적 거버넌스 역량을 비교 분석하는 학술 연구.
SAPD(State-based Adaptive Policy Design) 이론 + 실증 검증을 **통합 1편**으로 한국행정학보 투고 예정.

### 3.1 ACW 최종 총점

*잠정 평가 (순수I 4건/T+I 6건). 음수 0건은 시간 부족 가능성. 임기 완료 후 재평가 필요.

※자초적 임기 단축(윤석열·박근혜)은 내재적 벌칙. 연평균 정규화 부적절.

| 역할 | 기간 | ACW | 음수 | AI 검증 | 특징 |

## 4. 방법론 혁신

### 4.1 분석 프레임워크 (재구조화 반영)

**본 연구는 체계적 탐색 분석(systematic exploratory analysis)이며, 강한 인과적 비교를 주장하지 않는다.**

```
[핵심 3축]
1. 제도 분기점 분석 — 어떤 법률/제도가 언제 만들어졌는가
2. T/I/D/E 귀속 프로파일 — 각 대통령의 실제 기여 패턴
3. Lowi 정책 설계 분석 — 적응적 도구 채택 (엄격 기준)

[보조 지표]
- ACW 차원별 프로파일 (총점 순위는 보조)
- WGI 디트렌딩 (탐색적 패턴)
- AI 행위 방향성 검증 (보조 도구)

[검증 장치]
- Blind AI Validation (⚠️ 라벨 버그 발견, 재실행 대기)
- 1사건-1기준 원칙 (이중계상 해소)
- T+I 정량화 규칙
- Sensitivity Analysis
```

### 4.2 축적 편향 해결

- **Stock-Flow 이중 보고**: 제도 총량(stock)과 순기여(flow) 분리
- **WGI 디트렌딩**: 선형 추세 +0.026/년 제거 후 잔차 비교
- **OECD DiD**: 38개국 평균 대조군으로 글로벌 추세 통제
- **시대 상대 평가**: 절대 평가 + 시대적 과제 대응 + 선제적 기여 3관점

### 4.3 정규화 원칙

- 총점이 주 지표 (ACW = 축적적 역량)
- 연평균은 보조 참고만
- 자초한 임기 단축 = 내재적 벌칙 (추가 정규화 불필요)
- 현직 정부 = 잠정 평가 표기
