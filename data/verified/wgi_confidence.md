## 용어 변경 공지

> **"순기여(net contribution)"는 인과적 귀속을 암시하므로 부적절하다.**
> 본 문서부터 다음과 같이 용어를 변경한다:
>
> | 구 용어 | 신 용어 | 사유 |
> |---------|---------|------|
> | 순기여 (net contribution) | **추세제거 잔차 (detrended residual)** | 잔차 ≠ 인과적 기여. 추세 제거 후 남은 잔여분일 뿐 대통령의 인과적 효과로 귀속 불가 |
> | 순기여 분리 | **잔차 추정 (residual estimation)** | 분리(decomposition)는 인과적 식별을 함축하나, 실제로는 추세 적합 후 잔차 산출 |
>
> 기존 파일(`tide_attribution.md`, `final_results.md`, `acw_scoring_final.md` 등)에서도 동일하게 변경 필요.

## 2. 추세제거 잔차(detrended residual) + 신뢰구간

### 추세 모형

- **선형 추세**: GE.EST = 0.026 * (Year - 2000) + 0.614 (R^2 = 0.73)
- 추세 기울기: +0.026/year

### 대통령별 잔차 추정 및 95% CI

잔차 = (임기말 GE.EST - 임기초 GE.EST) - (추세 기울기 x 재임 연수)

잔차의 SE는 임기초·임기말 두 추정치의 결합 표준오차로 계산:
- SE_diff = sqrt(SE_start^2 + SE_end^2) (독립 가정)
- 95% CI = 잔차 +/- 1.96 * SE_diff

| 대통령 | 기간 | 실제 Δ | 추세 Δ | **추세제거 잔차** | SE_diff | 95% CI | 0 포함? |
|--------|------|-------:|-------:|-----------:|--------:|-------:|:-------:|
| 노무현 | 2003-2007 | +0.412 | +0.104 | **+0.308** | 0.279 | [-0.239, +0.855] | **Yes** |
| 이명박 | 2008-2012 | +0.081 | +0.130 | **-0.049** | 0.301 | [-0.639, +0.541] | Yes |
| 박근혜 | 2013-2016 | +0.018 | +0.104 | **-0.086** | 0.294 | [-0.662, +0.490] | Yes |
| 문재인 | 2017-2021 | +0.337 | +0.130 | **+0.207** | 0.314 | [-0.409, +0.823] | **Yes** |
| 윤석열 | 2022-2023 | +0.056 | +0.026 | **+0.030** | 0.358 | [-0.672, +0.732] | Yes |

**핵심 결론: 모든 대통령의 추세제거 잔차는 95% CI에서 0을 포함한다.**

## 3. 대통령 간 잔차 차이의 유의성 검정

두 대통령 간 잔차 차이의 SE = sqrt(SE_diff_A^2 + SE_diff_B^2)

가장 큰 잔차 차이: 노무현(+0.308) vs 박근혜(-0.086) = 0.394

| 비교 | 잔차 차이 | SE_combined | z-score | p-value | 유의? (p<0.05) |
|------|----------:|------------:|--------:|--------:|:--------------:|
| 노무현 vs 박근혜 | 0.394 | 0.405 | 0.97 | 0.331 | **No** |
| 노무현 vs 이명박 | 0.357 | 0.410 | 0.87 | 0.384 | **No** |
| 문재인 vs 박근혜 | 0.293 | 0.430 | 0.68 | 0.496 | **No** |
| 문재인 vs 이명박 | 0.256 | 0.435 | 0.59 | 0.555 | **No** |
| 노무현 vs 문재인 | 0.101 | 0.420 | 0.24 | 0.810 | **No** |

**핵심 결론: 어떤 대통령 쌍의 잔차 차이도 통계적으로 유의하지 않다.**

WGI의 측정 불확실성(SE ~ 0.20)이 너무 커서, 대통령 임기 간 잔차 차이(최대 ~ 0.39)를 유의미하게 구별할 수 없다.

## 4. 감도 분석: WGI를 적응 역량 프로파일에서 제거하면?

### 현재 ACW에서 WGI가 기여하는 경로

WGI는 다음 ACW 차원에서 근거로 사용됨:
1. **Le3 (리더십 연속성/변화 대응)**: WGI 잔차를 리더십 효과의 보조 근거로 활용
2. **FG1 (공정한 거버넌스)**: WGI percentile을 국제 비교 근거로 활용
3. **T/I/D/E C1 코드**: 각 정부의 WGI 변화를 T+I 또는 I- 코딩의 근거로 활용

### WGI 제거 시 영향

| 항목 | WGI 포함 (현재) | WGI 제거 시 | 변화 |
|------|:-----------:|:----------:|:----:|
| **TIDE 코딩 변경** | C1 항목 존재 (6개 정부 각 1건) | C1 항목 삭제 | 각 정부 I 또는 I- 1건 감소 |
| **Net count 변동** | | | |
| - 노무현 | +7 | +6 (C1의 I 제거) | -1 |
| - 이명박 | +4 | +5 (C1의 I- 제거) | +1 |
| - 박근혜 | +2 | +3 (C1의 I- 제거) | +1 |
| - 문재인 | +15 | +14 (C1의 I 제거) | -1 |
| - 윤석열 | +1 | +1 (C1은 T로 코딩, Net 무변동) | 0 |
| **순위 변동** | 문>이재명>이명박>노무현>윤석열>박근혜 | 문>이재명>노무현=이명박>윤석열=박근혜 | **동률 발생, 순위 역전 없음** |
| **ACW 총점** | | Le3에서 WGI 근거 삭제, 나머지 근거 유지 | 총점 변동 미미 (WGI는 보조 근거) |

### WGI 제거 시 결론

1. **순위 불변**: WGI를 완전히 제거해도 상위(문재인, 이재명)와 하위(윤석열, 박근혜) 그룹 구분은 유지된다.
2. **중간 그룹 변동**: 노무현과 이명박 간 차이가 축소되어 동률 가능성이 발생한다.
3. **분석 권고**: WGI는 "탐색적 보조 지표"로서 포함하되, 인과적 귀속의 근거로 사용하지 않는 것이 적절하다. 신뢰구간 분석에서 대통령 간 차이가 유의하지 않으므로, WGI 잔차에 기반한 T/I/D/E 코딩은 근거가 약하다.

## 5. 종합 권고

### 5-1. WGI 사용에 대한 방법론적 한계

| 한계 | 설명 |
|------|------|
| **측정 불확실성** | SE ~ 0.20 (추정치 범위 0.6~1.4 대비 약 15-30%). 연간 변동(평균 ~ 0.05)보다 SE가 4배 이상 크다 |
| **인과 귀속 불가** | 잔차 ≠ 대통령 효과. 다른 비추세 요인(경제 위기, 글로벌 인식 변화 등) 미통제 |
| **시차 효과** | WGI는 전문가/서베이 인식 지표로, 정책 효과가 반영되기까지 1-3년 시차 존재 |
| **N=6** | 대통령 수가 극히 작아 통계적 추론이 구조적으로 불가능 |

### 5-3. 논문 기술 권고

WGI 디트렌딩 섹션에 다음을 반드시 기술할 것:

> "WGI Government Effectiveness의 추세제거 잔차(detrended residual)를 산출하였으나, 각 추정치의 표준오차(SE ~ 0.20)가 대통령 임기 간 잔차 차이(최대 0.39)와 동일 규모이다. 95% 신뢰구간에서 모든 대통령의 잔차는 0을 포함하며, 대통령 간 잔차 차이도 통계적으로 유의하지 않다(p > 0.33). 따라서 WGI 잔차는 탐색적 패턴 확인에만 활용하며, 대통령별 적응 역량의 인과적 차이를 입증하는 증거로 해석하지 않는다."

## 6. 부록: 계산 상세

### 6-1. SE_diff 계산

```
노무현: SE_diff = sqrt(0.185^2 + 0.209^2) = sqrt(0.0342 + 0.0437) = sqrt(0.0779) = 0.279
이명박: SE_diff = sqrt(0.216^2 + 0.209^2) = sqrt(0.0467 + 0.0437) = sqrt(0.0904) = 0.301
박근혜: SE_diff = sqrt(0.206^2 + 0.209^2) = sqrt(0.0424 + 0.0437) = sqrt(0.0861) = 0.294
문재인: SE_diff = sqrt(0.211^2 + 0.233^2) = sqrt(0.0445 + 0.0543) = sqrt(0.0988) = 0.314
윤석열: SE_diff = sqrt(0.248^2 + 0.258^2) = sqrt(0.0615 + 0.0666) = sqrt(0.1281) = 0.358
```

### 6-2. 노무현 vs 박근혜 z-test

```
잔차 차이 = 0.308 - (-0.086) = 0.394
SE_combined = sqrt(0.279^2 + 0.294^2) = sqrt(0.0778 + 0.0864) = sqrt(0.1642) = 0.405
z = 0.394 / 0.405 = 0.973
p = 2 * (1 - Phi(0.973)) = 2 * 0.1654 = 0.331
```

### 6-3. 이명박 잔차 보정

기존 문서에서는 이명박 잔차를 -0.023으로 기재했으나, 정밀 계산으로 보정:
- 실제 Δ = 1.075 - 0.994 = 0.081
- 추세 Δ = 0.026 * 5 = 0.130 (2008-2012)
- 잔차 = 0.081 - 0.130 = **-0.049**
- 기존 -0.023과의 차이는 추세 시작점/끝점 정의 차이에 기인 (미미한 차이)
