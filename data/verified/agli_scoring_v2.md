## 1. 이론적 기반: Ostrom IAD Framework

### Rules-in-Form vs. Rules-in-Use

Ostrom (2005, *Understanding Institutional Diversity*)은 공식적 법규(rules-in-form)와 실제 행위자가 따르는 운영 규범(rules-in-use) 간의 괴리를 제도 분석의 핵심으로 제시하였다. 법률의 존재 여부만으로 채점하면 rules-in-form만 반영되어 변별력이 제한된다. 예컨대:

- D1(정책 피드백): 법률 존재만으로 채점 시 전 정부 동일 점수 (변별력 0)
- D3(시민참여): 법률 존재만으로 채점 시 전 정부 동일 점수 (변별력 0)

AGLI는 법률 존재(rules-in-form) + 실제 작동 수준(rules-in-use)을 결합한 0-4 스케일로 설계하여, 동일 법률 하에서도 정부별 운영 차이를 포착한다.

### 명칭 및 이론적 위치

- **AGLI** (Adaptive Governance Legal Infrastructure, 적응적 거버넌스 법제도 지표)
- Ostrom IAD 프레임워크 기반의 rules-in-form + rules-in-use 통합 측정

## 2. 0-4 스케일 설계

### 공통 프레임워크

| 점수 | 수준 | 정의 |
|------|------|------|
| **0** | 법 부존재 | 해당 차원의 공식 제도(법률/기구)가 존재하지 않음 |
| **1** | 법 존재 + 미작동 | 법률이 존재하나 실질적으로 작동하지 않거나 거버넌스 붕괴 상태 |
| **2** | 법 존재 + 제한적 작동 | 법률 존재, 형식적 절차는 이행하나 실질적 성과/환류 미흡 |
| **3** | 법 존재 + 보통 작동 | 법률 존재, 절차 이행 + 측정 가능한 실질적 성과 일부 확인 |
| **4** | 법 존재 + 체계적 작동 | 법률 존재 + 체계적 운영 + 측정 가능한 성과와 환류 작동 |

---

### D1: 정책 피드백 (Policy Feedback) — 차원별 기준

| 점수 | 기준 | 조작적 정의 |
|------|------|------------|
| 0 | 평가 법률 부재 | 정부업무평가 관련 법적 근거 없음 |
| 1 | 법 존재 + 미작동 | 법률 존재하나 거버넌스 붕괴(탄핵/계엄 등)로 평가-환류 정상 작동 불가 |
| 2 | 법 존재 + 형식적 작동 | 매년 평가 실시, 결과 공개/국회보고는 이행하나 예산 반영 약 1% 수준, 실질 환류 미흡 |
| 3 | 법 존재 + 추가 환류 채널 | 기본법 운영 + 추가 평가/환류 법률(데이터산업법, 이태원특별법 AAR 등) 도입으로 다층적 피드백 |
| 4 | 법 존재 + 실질 환류 작동 | 평가 결과의 예산/인사/조직 반영이 체계적으로 문서화되고 측정 가능 |

### D2: 정책 실험 (Policy Experimentation) — 차원별 기준

| 점수 | 기준 | 조작적 정의 |
|------|------|------------|
| 0 | 제도 부재 | 규제샌드박스/특구 법률 미존재 |
| 1 | 법률 추진 실패 | 관련 법안 발의/추진했으나 국회 통과 실패 |
| 2 | 제도 도입 초기 | 법률 시행 + 승인 건수 200건 미만 또는 규제개선 완료율 매우 낮음 |
| 3 | 제도 본격 운영 | 법률 시행 + 누적 승인 500건+ 또는 특구 20개+ 지정 + 규제개선 일부 진행 |
| 4 | 체계적 성과 | 누적 승인 1,000건+ + 규제개선 완료율 30%+ + 다분야 확산 |

### D3: 시민참여 (Citizen Participation) — 차원별 기준

| 점수 | 기준 | 조작적 정의 |
|------|------|------------|
| 0 | 채널 부재 | 공식적 시민참여 온라인 채널 없음 |
| 1 | 기본 채널만 존재 | 전자정부법 제17조 기반 일반 민원/신문고 수준, 정책 반영 경로 불명확 |
| 2 | 채널 확대 시도 | 신규 참여 채널 도입 시도했으나 이용률 매우 낮거나(방문자 0.7% 수준), 기존 채널 폐지로 후퇴 |
| 3 | 활발한 채널 운영 | 대규모 시민참여 채널 운영(수십만 건+), 정부 응답 체계 존재, 다만 법적 근거 미비 |
| 4 | 법적 근거 + 체계적 운영 | 법률에 근거한 시민참여 채널 + 대규모 이용 + 정책 반영 문서화 |

### D4: AI/디지털 거버넌스 (Digital Governance) — 차원별 기준

| 점수 | 기준 | 조작적 정의 |
|------|------|------------|
| 0 | 전자정부 법제 부재 | 전자정부 관련 법률/인프라 없음 |
| 1 | 기본 전자정부 | 전자정부법 존재 + EGDI 10위권 이하 또는 이용률 데이터 미비 |
| 2 | 전자정부 성숙 | 전자정부법 + EGDI 5위 이내 + 이용률 상승 추세 |
| 3 | 디지털 거버넌스 확장 | EGDI 3위 이내 + 데이터/디지털 관련 추가 법률 + 이용률 89%+ |
| 4 | AI 거버넌스 체계 구축 | AI기본법 시행 + 전담 위원회/기구 구성 + EGDI 상위권 유지 |

- **근거**: inst_operation_data.md "매년 실시, 120대 국정과제 기반 평가"
- **2점 근거**: 이태원특별법 AAR 등 추가 환류 채널은 있으나(3점 요건 일부 충족), 계엄 선포(2024.12.3)와 직무정지로 임기 후반 거버넌스가 심각하게 훼손됨. 박근혜 사례와 유사하게 rules-in-use 훼손이 확인되나, 임기 전반부(2022-2024 중반)에는 정상 운영되었으므로 1점까지는 하락하지 않음

- **근거**: inst_operation_data.md "누적 1,011건+ 달성, 8-9차+ 추가, 모빌리티/순환경제 확대"

- **근거**: inst_operation_data.md "국민청원 폐지, 국민제안 이용률 0.7%, 청원24 제약"

## 6. 방법론적 주석

### Ostrom IAD 적용 방식
1. **Rules-in-form**: 각 차원의 법률 존재 여부 확인
2. **Rules-in-use**: 법률의 실제 작동 수준을 운영 데이터로 평가
3. **Gap 분석**: form-use 괴리가 클수록 낮은 점수 부여 (예: 박근혜 D1 — 법 존재하나 거버넌스 붕괴)

### 한계
1. **D1 예산 반영 데이터**: "약 1% 수준 구조조정"은 2017년 기준 단일 데이터포인트로, 정부별 비교에 한계
2. **D2 규제개선 완료율**: 17.2%는 2019-2021년 198건 심층분석 기준으로, 정부별 분리 어려움
3. **D3 이재명 정부**: 취임 1년 미만으로 시민참여 채널 운영 실적 판단 제한
4. **D4 EGDI**: 2년 주기 발표로 정부 임기와 완전히 일치하지 않음
