# 5건 의사결정 전문가 패널 전문 (2026-03-27)

---

## Decision 1: WGI 기준 시점 → A (최신 API) 만장일치

- 방법론: 소급 개정이 시계열 비교 가능성 확보 목적. 원본은 연도별 방법론 차이로 비교 불가
- 행정학: 심사자는 일관된 데이터셋 미사용 시 "엄밀성 부족" 지적
- 데이터: 최신 API = single vintage, PDF 추출 = vintage mixing + OCR 오류
- 출판: 세계은행 자체가 "최신 DB 사용 권장" 명시
- 통계: 소급 개정 데이터가 standard error 더 작고 정밀

## Decision 2: OECD 평균 WGI → A (수집) 만장일치

- 방법론: OECD 동질적 비교군이 200국 Percentile보다 설득력 높음
- 행정학: "OECD 평균 대비"는 한국 행정학 관용적 프레임
- 데이터: 38국 × 24년 = ~900건, 자동화 5분
- 출판: 한국/OECD 추이 그래프는 시각적 임팩트
- 통계: 고정 패널 평균이 Percentile보다 안정적 대조군

## Decision 3: 노무현 국정과제 → B (세부과제 수집 시도) 만장일치, 조건부

- 12개 vs 120개 비율 비교는 통계적 무의미 (n=12일 때 0%의 95% CI = [0%, 26.5%])
- 6개 정부 > 5개 정부 (연구 범위, 제목 임팩트)
- 수집 기한 1주, 실패 시 A안 + 한계 명시 + 로버스트니스 체크

## Decision 4: AI 라운드 수 → B (3라운드 × 3 Tasks) 만장일치

- 검증자 역할에는 모드(최빈값) 추출 충분 → 3라운드 적절
- 5모델 × 3라운드 × 3 Tasks = 45회 독립 판단
- Task 세분화가 동일 질문 5회 반복보다 심사자에게 설득력 있음
- Temperature 0.0~0.3 고정, Krippendorff's alpha 보고

## Decision 5: 블라인드 실패 대응 → A (사전 의사결정 트리) 만장일치

사전 프로토콜:
1. 식별률 측정 → 16.7% 대비 이항검정
2. 식별 vs 비식별 항목 간 T/I/D/E 분포 χ² 검정
3. 유의미한 차이 없음 → 전체 사용 (A) + 식별률 투명 보고
4. 유의미한 차이 있음 → 비식별만 사용 (B) + 검정력 보고
