## 1. 학술지 편집장 의견

### 분리가 정당한 핵심 이유
- 연구 질문이 본질적으로 다름 (이론 vs 실증)
- 방법론이 완전히 다름 (Jaakkola 개념적 분석 vs Multi-LLM 정량 코딩)
- 통합본 §3의 59행 압축은 방법론적 엄밀성 훼손

### 실질적 위험 요소
1. 이론 논문의 공허함 (심각도: 중상) — SAPD에 실증 예시 없으면 "분석적 유용성 판단 불가" 지적 불가피
2. 실증 논문의 이론적 기반 부재 (심각도: 중) — SAPD에 의존하면 미출판 이론 전제 문제
3. 다중-LLM 방법론에 대한 회의적 반응 (심각도: 상) — ICC, 모델 간 합의도, 인간 전문가 비교 필수

### 권고사항
- 두 논문의 독립성 실질 확보
- 상호 참조 1-2회 이내 제한
- 투고 전략 분산 (같은 학술지 동시 투고 금지)

## 2. 방법론 전문가 의견

### Multi-LLM 스코어링 독립 기여 판단: 조건부 긍정
- 방법론적 신규성 실재 (5개 LLM 합의로 정부 역량 평가 선례 없음)
- 59행 압축은 방법론적으로 용납 불가 → 분리의 결정적 근거

### 취약점 2가지
1. 인간 코더 벤치마크 부재 — 단일 코더 비교는 수렴 타당도가 아닌 순환 논증
2. LLM 한국 정치 지식 편향 미통제 — Grok(+1.56) vs GPT(+0.73) 차이 원인 분석 필요

### 최종 권고: 분리 (3가지 수정 전제)
1. 인간 코더 벤치마크 또는 체계적 괴리 분석 추가
2. 논문 2 논의절에서 SAPD 1-2단락 참조 (이론-실증 연결)
3. 이재명 정부 약점 강하게 서술 (정치적 편향 방어)

## 3. 한국 행정학 교수 의견 (통합 강력 권고)

### 논문 2 (실증) 핵심 문제
1. **정치적 민감성** — 현직 대통령 평가 논문의 심사자 배정 자체 곤란
2. **결과가 너무 깔끔** — 이재명 ACW +1.12 압도적 1위, "AI가 현 정부 역대 최고" 결론
3. **임기 1년차 평가의 근본적 한계**
4. **다중 AI 방법론** — 해외 LLM-as-coder는 텍스트 코딩, 본 연구는 제도적 역량 질적 점수화 (성격 다름)
5. **인간 코더 벤치마크 부재** — 리스크가 아니라 치명적 결함

### 권고: 통합 1편 (방안 A)
- v6를 27,000~30,000자로 압축 + 부록 별도
- 이론-실증 균형 유지
- 실증 범위를 3개 정부(노무현, 문재인, 이재명)로 축소 검토
- 또는 국제학술지(GIQ 등) 영문 투고 검토

## 4. 출판 전략가 의견

### 현재 순서(논문 2 먼저) 평가: 차선, 최적 아님
- 논문 2에서 "이석민(2026a)" 참조 시 미출판 논문 의존 문제
- 교차참조를 사실상 제거하면 독립성 확보 가능

### 학술지 권고
- 논문 2: **정보화정책** 1순위 (AI 방법론에 개방적, 심사자 적합성)
- 한국정책학회보보다 리젝 시 시간 손실 적음

### Salami slicing 위험: 낮음
- 연구 질문·방법론 충분히 다름
- 서론 본문(각주 아님)에 프로젝트 출처 명시

### 단일 논문 + 온라인 부록 옵션
- 한국행정학보는 온라인 보충자료 시스템 미정착 → 비권장
- 업적 1편 vs 2편 차이는 교원업적평가에서 무시 불가

## 5. 반대론자 (Devil's Advocate) 의견

### 분리 반대의 최강 논거

1. **"왜 SAPD가 필요한가"의 설득력 치명적 훼손**
   - v6의 핵심 논증: "6개 정부 실증 비교 → 리더 의존성 발견 → SAPD 필요"
   - §3 제거 시 이 논증 체인 단절

2. **"독자층이 다르다"는 논거의 허구성**
   - 한국 행정학계 규모에서 "알고리즘 거버넌스"와 "비교 정부론" 독자는 사실상 동일

3. **분량 제한은 극복 가능**
   - v6 본문 ~28,000자 (부록 33,500자 별도)
   - 1,000자만 줄이면 27,000자 원칙 충족
   - 35,000자 상한 활용 가능

4. **두 편의 "약한 논문"보다 한 편의 "강한 논문"**
   - 분리 후 각 논문의 약점이 더 부각됨
   - 통합 시 이론-실증 선순환 유지

5. **분리의 숨겨진 비용**
   - Plan.md Phase A-D = 사실상 2편 새로 쓰기 (3-4배 비용)
   - 교차참조 관리의 함정

### 제3의 선택지 제안: 통합 논문 + 온라인 부록
- 본문 27,000자: 서론+이론+방법+SAPD+실증핵심+정합성+논의+결론
- 온라인 부록(GitHub): 부록 A-D 전체 + 원자료 + 코드

## 통합 결정 근거 (최종)

1. 본문 ~28,000자 → 편집으로 27,000자 가능 (분량 문제 아닌 편집 문제)
2. 이론-실증 선순환이 분리 시 파괴됨
3. 한국 학계에서 순수 이론 논문 게재 어려움
4. SAPD 이론이 정치적 민감성의 방어막 역할
5. 비용 효율: v6 편집 vs 2편 새로 쓰기

### 전원 동의 보완 사항
- 인간 코더 벤치마크 부재 → 체계적 한계 명시 또는 최소 부분 벤치마크 확보
- 이재명 정부 약점 부각 → 정치적 편향 방어
- LLM 방법론 정당화에 충분한 지면 할애
