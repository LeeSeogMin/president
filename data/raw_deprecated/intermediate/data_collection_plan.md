# 데이터 재수집 실행 계획

> 작성일: 2026-03-26
> 배경: 기존 data/raw/ 데이터가 AI 생성(fabricated)으로 판명. 전문가 5인 조사 결과 기반 재수집 계획.
> 상위 문서: context.md, plan.md

## 0. 기존 데이터 진위 판정 요약

| 데이터 | 판정 | 주요 오류 |
|--------|------|----------|
| 국제지표 (WGI, DGI, EGDI) | **거짓** | WGI 수치 전체 불일치 (2023: +1.16→+1.405), DGI 완전 오류 (0.64→0.935) |
| Policy Lag 날짜 | **부분 거짓** | 5건 사실 오류 (법률번호, 날짜, 특별법 존재 여부) |
| ACW 22기준 점수 | **AI 생성** | 저자도 AI도 1차 자료 확인 없이 생성 |
| INST 4차원 점수 | **AI 생성** | 동일 |
| 국정과제 분류 | **부분 거짓** | 과제 수부터 불일치 (이재명 정부 실제 123대) |
| 학술 인용 | **3건 허위** | 손호중(2007), 조성한(2011), 정재호(2015) — 주제 불일치 또는 미존재 |

## 1. Phase 1: 즉시 수집 (API/스크래핑, 인증 불필요)

### 1-1. WGI 시계열 (World Bank REST API)

| 항목 | 내용 |
|------|------|
| **데이터** | Government Effectiveness 추정치 + 백분위, 2000-2023 |
| **API** | `https://api.worldbank.org/v2/country/KOR/indicator/GE.EST?format=json&date=2000:2025` |
| **도구** | Python requests 또는 WebFetch |
| **인증** | 불필요 |
| **소요시간** | ~1분 |
| **산출물** | `data/verified/wgi_korea.csv` |
| **구현** | `data/scripts/data_pipeline.md` §1 참조 |

추가 수집 가능 지표: RQ.EST(규제 품질), RL.EST(법의 지배), VA.EST(시민 참여), CC.EST(부패 통제)

### 1-2. OECD DGI (SDMX API + PDF)

| 항목 | 내용 |
|------|------|
| **데이터** | Digital Government Index 6차원 + 종합, 2019/2023/2025 |
| **API** | `https://sdmx.oecd.org/public/rest/v2/data/dataflow/OECD.GOV.GIP/DSD_GOV@DF_GOV_DGOGD_2025/1.0/?format=csvfilewithlabels` |
| **PDF** | `https://www.oecd.org/content/dam/oecd/en/publications/reports/2024/01/2023-oecd-digital-government-index_b11e8e8e/1a89ed5e-en.pdf` |
| **도구** | Python requests (API) 또는 firecrawl (PDF 파싱) |
| **인증** | 불필요 |
| **소요시간** | ~2분 |
| **산출물** | `data/verified/oecd_dgi_korea.csv` |
| **주의** | DGI는 3회(2019, 2023, 2025)만 실시. 연속 시계열 아님 |

### 1-3. UN EGDI (Data360 API)

| 항목 | 내용 |
|------|------|
| **데이터** | E-Government Development Index, 격년 (2003-2024) |
| **API** | `https://data360api.worldbank.org/data360/data?dataset=UN_EGDI&country=KOR` |
| **대안** | `https://publicadministration.un.org/egovkb/en-us/Data-Center` (브라우저) |
| **도구** | Python requests (Data360) 또는 Chrome 자동화 (UN 사이트) |
| **인증** | 불필요 |
| **소요시간** | ~2분 |
| **산출물** | `data/verified/un_egdi_korea.csv` |
| **주의** | 격년 발행. 이재명 정부 데이터는 2026 조사에 포함될 예정 |

### 1-4. 법률 원문 (국가법령정보센터)

| 항목 | 내용 |
|------|------|
| **데이터** | AI 기본법, 규제샌드박스법, 정부업무평가기본법, 이태원 특별법 등 |
| **URL 패턴** | `https://www.law.go.kr/법령/{법령명}` |
| **도구** | firecrawl (JSON 모드, 테스트 완료) |
| **인증** | 불필요 |
| **소요시간** | 법령당 ~30초 |
| **산출물** | `data/verified/laws/` 폴더에 법령별 JSON |
| **추출 항목** | 법률번호, 공포일, 시행일, 제정기관, 주요 조문 |

수집 대상 법령:
- [ ] 인공지능 발전과 신뢰 기반 조성 등에 관한 기본법 (AI 기본법)
- [ ] 규제자유특구 및 지역특화발전특구에 관한 규제특례법
- [ ] 정부업무평가 기본법
- [ ] 10·29 이태원 참사 피해자 권리보장과 진상규명 및 재발방지를 위한 특별법
- [ ] 데이터 기본법
- [ ] 전자정부법

### 1-5. Policy Lag 날짜 검증 (정책브리핑 + 법제처)

| 항목 | 내용 |
|------|------|
| **데이터** | 6개 위기 사례의 인지/결정/실행 날짜 |
| **도구** | firecrawl search (`site:korea.kr` + 키워드) |
| **인증** | 불필요 |
| **소요시간** | 사례당 ~5분 |
| **산출물** | `data/verified/policy_lag_verified.md` |

검증 대상 (기존 5건 오류 수정 포함):
- [ ] 2008 금융위기 — 기획재정부 긴급회의 날짜

- [ ] 2020 코로나19 — 위기경보 격상 날짜
- [ ] 2020 재난기본소득 — 경기도 결정/집행 날짜
- [ ] AI 기본법 — 법률번호(제20676호), 공포일(2025.01.21)
- [ ] 2022 이태원 — 애도 선포(10.30), 특별법(2024.05.21 시행)

## 3. Phase 3: 데이터 재구축 (수작업 + AI 보조)

### 3-1. 국정과제 Lowi 유형 재분류

| 항목 | 내용 |
|------|------|
| **입력** | 6개 정부 595개 국정과제 실제 목록 (`data/raw/국정과제_수집_가이드.md`) |
| **방법** | 과제명 + 세부 실천과제 기반 Lowi 4유형 분류 |
| **코더** | 저자 직접 분류 + AI 보조 (교차 확인용) |
| **산출물** | `data/verified/lowi_classification.csv` |
| **주의** | 노무현 정부(12대)와 타 정부(100~140대)의 단위 차이 → 하위 과제 포함 여부 결정 필요 |

### 3-2. ACW 재코딩

| 항목 | 내용 |
|------|------|
| **전제 조건** | Phase 1에서 수집된 1차 자료 (법률, 감사보고서, 보도자료) 확보 |
| **방법** | Gupta et al.(2010) 원 프레임워크의 22기준을 근거 문서 기반으로 재점수화 |
| **코더** | 저자 직접 코딩 (근거 문서 URL 반드시 첨부) |
| **검증** | AI 5모델 재평가 (기존 스크립트 활용) + 단일코더-AI 괴리 분석 |
| **산출물** | `data/verified/acw_scoring_verified.md` |
| **핵심 원칙** | 모든 점수에 1차 자료 출처(URL 또는 문서명+페이지) 명시 |

### 3-3. INST 재코딩

| 항목 | 내용 |
|------|------|
| **방법** | Phase 1에서 수집된 법률 원문 기반으로 4차원 재점수화 |
| **근거** | 법률 존재 여부 → 0/1/2 점수 부여 (객관적 기준) |
| **산출물** | `data/verified/inst_scoring_verified.md` |

## 7. 미해결 의사결정 사항

| # | 사항 | 선택지 | 영향 |
|---|------|--------|------|
| 1 | WGI 기준 시점 | (a) 최신 API 값 (소급 개정 반영) vs (b) 해당 연도 원본 보고서 값 | 정부별 Δ 수치 변동 |
| 2 | 노무현 정부 국정과제 단위 | (a) 12대 상위 과제만 vs (b) 하위 세부과제 포함 | Lowi 분류 비율 비교 가능성 |
| 3 | ACW 코딩 방식 | (a) 저자 단독 → AI 검증 vs (b) 인간 전문가 2-3명 패널 | 방법론적 신뢰도 |
| 4 | 이재명 정부 국제지표 부재 | (a) "데이터 없음" 표기 vs (b) 해당 분석 제외 | 6개 정부 비교의 완전성 |
