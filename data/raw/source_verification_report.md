- **실제 확인**: **2022.10.30** (사고 다음 날 오전 대통령실 발표)

- **출처**: MBC 뉴스, 법률신문 2024.01.09, 국가법령정보센터

- **실제 확인**: 5월 11일은 **증상 발현일**. 첫 의원 방문은 5월 12일, 평택성모병원 방문은 5월 15~17일
- **출처**: 위키백과 MERS 문서, 질병관리본부 역학조사 보고서
- **심각도**: 낮음 (문제 발생 기점으로서 5.11 사용은 합리적이나, "입원"이 아니라 "증상 발현"으로 수정 필요)

### [오류 7] MERS 추경 국회 통과일
- **현재 기재**: 2015.07.13
- **실제 확인**: **2015.07.24** (국회 본회의 통과)
- **출처**: KDI 경제정보센터, 위키백과 2015년 문서
- **심각도**: 중간 (추경까지 총 지연 일수 변경: 63일→74일)

### [오류 8] 경기도 재난기본소득 지급 시작일
- **현재 기재**: 2020.04.07
- **실제 확인**: 신용카드 신청 시작은 **4월 9일**, 선불카드는 4월 20일부터
- **출처**: 경기도 공식 뉴스 (gnews.gg.go.kr), 경기일보
- **심각도**: 낮음~중간 (4월 7일은 행정 집행 시작일일 가능성 있으나, 실질 지급 시작은 4월 9일)

### [오류 9] 국가AI전략위원회 위원 수
- **현재 기재**: "50인 구성"
- **실제 확인**: 초기 출범 시 **50명** (정부위원 13명 + 대통령실 2명 + 민간위원 34명 + 부위원장), 이후 분과위원 52명 추가 선임하여 **총 85명**
- **출처**: AI타임스, 정책브리핑
- **심각도**: 낮음 (50인은 본위원회 기준으로 정확, 단 분과위원 포함 시 85명)

## 4. 정부 문서 접근성 테스트 결과

### 접근 가능 (직접 스크래핑/검색 가능)

| 사이트 | URL | 접근 방법 | 비고 |
|--------|-----|----------|------|
| **정책브리핑** | https://www.korea.kr | WebFetch + firecrawl | 검색기간·기관별 필터 가능. URL: `/briefing/pressReleaseView.do?newsId=[ID]` |
| **국회 의안정보시스템** | https://likms.assembly.go.kr/bill/ | WebFetch + firecrawl | 기본검색, 문서검색, 지능형검색 3종 제공 |
| **국가법령정보센터** | https://www.law.go.kr | firecrawl (JSON 추출 우수) | 법률번호, 공포일, 시행일 등 구조화 추출 가능 |

### 접근 제한 (대안 필요)

| 사이트 | URL | 문제 | 대안 |
|--------|-----|------|------|
| **관보** | https://gwanbo.mois.go.kr | firecrawl 타임아웃/차단 | 국가법령정보센터에서 동일 정보 확인 가능 |
| **나무위키** | namu.wiki | 403 Forbidden (WebFetch) | firecrawl 또는 위키백과 대체 |
| **질병관리청** | https://www.kdca.go.kr | 일부 문서 PDF만 제공 | PDF 직접 다운로드 후 분석 |

### 권장 데이터 수집 파이프라인

```
1단계: 국가법령정보센터 (firecrawl JSON 추출)
  → AI 기본법 원문, 대통령령, 시행령 등 법률 문서
  → URL 패턴: law.go.kr/lsInfoP.do?lsiSeq=[시퀀스번호]

2단계: 정책브리핑 검색 (WebFetch)
  → 기관별·기간별 보도자료 검색
  → URL 패턴: korea.kr/briefing/pressReleaseView.do?newsId=[ID]

3단계: 국회 의안정보시스템 (WebFetch)
  → 추경안, 특별법 등 의안 처리 경과
  → URL 패턴: likms.assembly.go.kr/bill/billDetail.do?billId=[ID]

4단계: 학술DB 교차검증
  → KDI 경제정보센터, 국회예산정책처 보고서
```
