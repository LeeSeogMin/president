## 1. SAPD 핵심 논제

**연구 질문**: 증거 기반 정책(EBP)의 구조적 한계를 극복하는 새로운 정책 패러다임으로서, SAPD는 어떤 이론적 기초와 설계 아키텍처를 통해 실시간 적응형 거버넌스를 구현할 수 있는가?

**핵심 공식**: `State(t) → Decision(t) = f(State(t)) → System Response → State(t+1) → [반복]`

### 3대 명제

| 명제 | 내용 |
|------|------|
| P1 | EBP의 시간 지연은 구조적 아키텍처 결함 → 패러다임 수준 대응 필요 |
| P2 | 정책 시스템은 상태 전이 함수로 모델링 가능 → 연속적 적응 구현 |
| P3 | SAPD 5단계 아키텍처가 적응형 정책 구현의 충분한 설계 명세 제공 |

### SAPD 5-Layer

| Layer | 이름 | 기능 |
|-------|------|------|
| L1 | 문제 구조화 | 상태 공간 정의 |
| L2 | 전략 구조화 | 목표 함수, 전략 대안, 평가 기준 |
| L3 | 의사결정 아키텍처 | 의사결정 규칙, 파라미터, 인간 개입 조건 |
| L4 | 데이터 통합 | 실시간 상태 관측, AI 통합 |
| L5 | 평가 및 적응적 피드백 | 지속적 격차 측정, 3단계 적응 |

### 주요 발견

1. **진보-보수 거버넌스 분리**: 진보 정부가 거의 모든 방법에서 보수 정부 능가 (인간·AI 일관)
2. **문재인 ACW-INST 괴리**: 강한 제도 설계이나 유연성/다양성 부족
3. **윤석열 평가 난이도**: 모델간 불일치 최대 → 거버넌스 특성 모호
4. **이재명 Le3 결핍**: 전 역할·전 모델에서 협력적 리더십 일관 최저
